# 💳 Banking Fraud Detection System

An AI-powered fraud detection system that identifies fraudulent transactions in real-time for digital banking environments.

## 📌 Project Overview

This system uses machine learning to detect fraudulent credit card transactions with high accuracy and low latency. The model achieves 82.6% fraud recall with only 1.2% false positive rate, processing each transaction in under 50 milliseconds.

## 🎯 Key Features

- Real-time fraud detection with sub-50ms latency
- 30 engineered features for accurate fraud pattern recognition
- SHAP explainability for transparent decision making
- REST API for easy integration with banking systems
- Interactive dashboard for fraud analysts
- Model monitoring with drift detection

## 📊 Dataset

- **Source:** Credit Card Fraud Detection Dataset
- **Records:** 10,000 transactions
- **Features:** 10 (amount, hour, merchant, device score, location, etc.)
- **Fraud Rate:** 1.5% (150 fraud cases)
